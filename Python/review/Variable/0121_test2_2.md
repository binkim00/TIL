## 🧪 테스트 2-확장② — 전역 변수 “읽기만” 하는 경우

```python
count =5

def show():
    print(count)

show()

```

---

### ❓ 질문 1

`show()`를 실행했을 때 출력되는 값은 얼마인가?

### ✅ 답변

> 5이다.
> 

✔ **정답**

👉 함수 안에서 `count`를 변경하지 않고 읽기만 하므로, 함수 밖에 있는 전역 변수 `count`의 값이 그대로 출력된다.

---

### ❓ 질문 2

이 코드에는 `global count`가 없는데도 왜 에러가 발생하지 않는가?

### ✅ 답변

> 함수 밖에 count가 선언되어 있어서 전역 변수의 값을 사용하기 때문이다.
> 

✔ **방향은 맞다. 표현을 정리한다.**

### ✅ 정확한 개념

> 함수 안에서 count를 읽기만 했기 때문에,
> 
> 
> 파이썬은 전역 변수 `count`를 그대로 사용해도 된다고 판단한다.
> 

👉 전역 변수는 **읽기만 할 경우** `global` 선언이 필요 없다.

---

### ❓ 질문 3

이 코드에서 `count`는 전역 변수인가, 지역 변수인가?

### ✅ 답변

> 전역 변수이다.
> 

✔ **정답**

👉 `count = 5`는 함수 밖에서 선언되었으며, 함수 안에서 새로 대입되지 않았으므로 전역 변수이다.

---

### ❓ 질문 4

함수 안을 아래와 같이 바꾸면 어떤 일이 발생하는가?

```python
def show():
    print(count)
    count +=1

```

### ❌ 답변

> 정상 실행된다. count에 값을 추가하는 것은 이상이 없다.
> 

⚠️ **오답**

---

### ✅ 정확한 개념

> 이 코드는 실행 시 에러가 발생한다.
> 

이유는 다음과 같다.

- `count += 1`이 존재하면
    
    파이썬은 이 함수 안에서 `count`를 **지역 변수로 만들려고 한다고 판단한다.**
    
- 그러나 `print(count)`에서
    
    **아직 생성되지 않은 지역 변수 `count`를 먼저 읽으려고 하므로** 에러가 발생한다.
    

👉 읽기와 쓰기가 섞이면, 파이썬은 지역 변수로 판단한다.

---

### ❓ 질문 5

전역 변수를 사용할 때 에러가 나는 경우와 나지 않는 경우의 차이는 무엇인가?

### ❌ 답변

> 모르겠다.
> 

✔ **정확한 진단이다. 이 테스트의 핵심이다.**

---

### ✅ 정확한 정리

> 전역 변수는 함수 안에서 읽기만 하면 에러가 나지 않는다.
> 
> 
> 그러나 함수 안에서 **값을 변경하려고 하면**,
> 
> `global` 선언이 없을 경우 파이썬은 해당 변수를 지역 변수로 판단하여 에러가 발생한다.
> 

---

## 📌 테스트 2-확장② 핵심 요약 (md용)

```markdown
- 전역 변수는 함수 안에서 읽기만 할 경우 global이 필요 없다
- 전역 변수를 변경하려 하면 global 선언이 필요하다
- 함수 안에서 대입이 발생하면 해당 변수는 지역 변수로 판단된다
- 지역 변수를 만들기 전에 읽으려고 하면 에러가 발생한다

```

## 📌 전역 변수와 지역 변수 핵심 정리

### 전역 변수(Global Variable)

- 함수 **밖**에서 선언된 변수이다
- 프로그램 전체에서 접근 가능하다
- 함수 안에서 **읽기만** 할 경우 `global` 선언이 필요 없다
- 함수 안에서 **값을 변경**하려면 `global` 선언이 필요하다
- 함수 실행이 끝나도 값이 **유지된다**

---

### 지역 변수(Local Variable)

- 함수 **안**에서 선언된 변수이다
- 해당 함수 안에서만 사용 가능하다
- 함수 실행이 끝나면 **사라진다**
- 함수 안에서 대입(`=`)이 발생하면 지역 변수로 판단된다

---

### 전역 변수 사용 시 에러 발생 기준

```markdown
- 전역 변수를 읽기만 하면 에러가 나지 않는다
- 전역 변수를 함수 안에서 변경하려고 하면 global이 필요하다
- global 없이 값을 변경하려 하면 파이썬은 지역 변수로 착각한다
- 지역 변수를 만들기 전에 읽으려고 하면 에러가 발생한다

```
